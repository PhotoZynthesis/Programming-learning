## 1. 概述

- XML（Extensible Markup Language），可扩展标记语言。可扩展指标签都是自定义的。
- XML的功能：
  - 配置文件
  - 在网络中传输



## 2. 语法

- **文档声明**：

  ```xml
  <?xml 属性='值' ...?>
  ```

  文档声明是XML的必须部分，而且必须置于文档开头。

  常用属性列表：

  - version：版本号，必须的属性
  - encoding：编码方式，默认IOS-8859-1
  - standalone：是否独立（依赖于其他文件），取值：
    - yes：依赖其他文件
    - no：不依赖其他文件

- **指令（结合CSS）**：

  ```xml
  <?xml-stylesheet type='text/css' href='demo.css'?>
  ```

- **标签**：

  自定义名称的标签。

  规则：

  - 名称可以包含字母、数字等字符
  - 不能以数字或标点符号开头
  - 不能以`xml`开头
  - 名称不能包含空格

- **属性**：

  注意事项：

  1. ID的属性值需要唯一。

- **文本**：

  - CDATA区：

    该区域中的数据会被原样展示，格式如下：

    ```xml
    <![CDATA[
    	...
    ]]>
    ```
